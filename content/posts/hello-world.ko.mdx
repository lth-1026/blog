블로그를 새로 시작했습니다. 글을 쓰면서 동시에 시스템도 다듬어 가려고 합니다.

## 왜 다시 만들었나

기존에 쓰던 노션은 빨랐지만 다음과 같은 한계가 있었습니다.

- 코드 하이라이팅과 커스텀 컴포넌트가 부족했다
- 검색엔진에 노출되는 형태가 제한적이었다
- 디자인을 내 손으로 조정할 수 없었다

<Callout type="info">
  이 글은 한국어 + 영어 두 버전으로 작성되어 있습니다. 푸터의 언어 토글로 전환할 수 있습니다.
</Callout>

## 기술 스택

```ts
const stack = {
  framework: "Next.js 16 (App Router)",
  language: "TypeScript 5",
  styling: "Tailwind v4 + shadcn/ui",
  content: "MDX + Shiki",
  hosting: "Vercel",
};
```

App Router의 RSC 덕분에 글 본문이 정적으로 사전 렌더링됩니다. 크롤러 입장에선 JS 실행 없이 바로 HTML을 받아갑니다.

## 앞으로

소소하게 배운 것들, 일하면서 겪은 문제들, 가끔은 만들고 싶은 것에 대한 메모를 올려두려고 합니다.
